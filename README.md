# apartment-management-system

## 概要
Spring Boot を用いた賃貸管理システムのバックエンド学習プロジェクトです。

## 使用技術
- Java
- Spring Boot
- MyBatis-Plus
- MySQL
- Redis
- MinIO
- Docker
- JUnit
- Mockito
- JWT
- Knife4j

## 主な学習内容
- CRUD API 実装
- ページング
- 認証認可
- 画像アップロード
- キャッシュ処理
- 単体テスト
- Docker による環境構築

## 補足
本プロジェクトは動画教材をもとに学習した内容を整理したものです。
